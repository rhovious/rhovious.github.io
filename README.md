# rhovious.github.io
rhovious.github.io

Clone

``` bash
git clone --recurse-submodules https://github.com/rhovious/rhovious.github.io.git


```


Update submodules after cloning
``` bash

git submodule update --init --recursive
```


Add submodules if they were not there
``` bash
$ git submodule add https://github.com/rhazdon/hugo-theme-hello-friend-ng.git themes/hello-friend-ng

$ git submodule add -f https://github.com/panr/hugo-theme-terminal.git themes/terminal

$ git submodule add https://github.com/schnerring/hugo-theme-gruvbox.git themes/gruvbox

```
