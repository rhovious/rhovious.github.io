Wedding Countdown with Magtag

Countdown to an event using the Adafruit Magtag.

About

A quick python script that I used to create a countdown to my wedding using the Adafruit Magtag.

This can also be used as a generic countdown to any event.

Hardware Required

  • Adafruit Magtag
    • Another CircuitPython board should work, but would require modification of the e-ink code.
  • 1S Lipo (Optional, for power)

Assembly

  • Install into 3D printed case (Optional)
  • Install liPo battery (Optional)
